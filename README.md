## 🛠️ How to Use This Architecture

To prevent gigantic, unreadable files, this repository utilizes an **Index/Example** structure.

1.  **Navigate to a Domain:** Click a link in the index above to enter a specific discipline (e.g., *Productivity / Strategy / OKR Goal Setting*).
2.  **The `index.md` File:** Every folder contains an `index.md` file. This houses the actual prompt logic, constraint rules, and variables you need to copy/paste into your AI.
3.  **The `eg.md` File:** Every folder contains an `eg.md` file charting real-world, high-quality examples of that prompt in action so you know what "good" looks like.
